# Apartment Management Web Application

This project is a simple full-stack web application designed for apartment building management.  
It includes a **React frontend** and a **Node.js (Express) backend** connected to an **Azure SQL Database**.

The application currently focuses on **user authentication (login)** and a **dashboard view** with static apartment-related data such as account balance, tasks, voting results, and reported problems.

---

## Features

- User login with credential verification
- Client-side routing using React Router
- Dashboard layout with sidebar navigation
- Backend authentication API
- Connection to Azure SQL Database
- CORS-enabled API for frontend communication

---

## Tech Stack

### Frontend
- React
- React Router DOM
- Fetch API
- CSS (custom styling)

### Backend
- Node.js
- Express.js
- MSSQL (Azure SQL Database)
- dotenv
- CORS
